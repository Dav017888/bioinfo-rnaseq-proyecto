# Proyecto de Bioinformática – Control de Versiones (UNIR)

Este proyecto es parte de la actividad grupal del Máster Universitario en Bioinformática (UNIR).
El objetivo principal es practicar el uso de Git y GitHub como herramientas de control de versiones.

## Objetivos del proyecto
- Crear y organizar un repositorio
- Aprender y practicar Git
- Usar ramas, commits y pull requests
- Documentar un proyecto en Markdown
